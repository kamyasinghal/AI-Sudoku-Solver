Algorithm Notes

BT
Backtracking

Recursive trial-and-error. Guaranteed to find a solution if one exists.

GR
Greedy

Fills naked singles first; greatly reduces backtracking on easy/medium puzzles.

DC
Divide & Conquer

Decomposes into 3×3 sub-problems before cross-box resolution.

DP
DP + MRV

Memoises constraint data; MRV picks the most constrained cell first.

HY
Hybrid

Chains Greedy → D&C → DP+MRV. Most efficient overall.
